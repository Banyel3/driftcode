# Code of Conduct

## Our Commitment

DriftCode is an open, welcoming project. Everyone participating — contributors, maintainers, and users — is expected to treat others with respect and professionalism.

## Standards

**Expected behavior:**
- Communicate constructively and respectfully
- Focus feedback on ideas and code, not people
- Welcome contributors of all experience levels
- Accept constructive criticism graciously
- Prioritize what is best for the community

**Unacceptable behavior:**
- Personal attacks or insults
- Dismissive or demeaning responses
- Deliberate intimidation or bullying
- Publishing others' private information without consent
- Any conduct that would be considered unprofessional in a collaborative setting

## Scope

This Code of Conduct applies in all project spaces — GitHub issues, pull requests, discussions, and any public representation of the project (social media, conferences, etc.).

## Enforcement

Violations can be reported to **cornelio.vaniel38@gmail.com**. All reports will be reviewed confidentially. Maintainers will take appropriate action, up to and including permanent removal from the project.

## Reference

This Code of Conduct is informed by the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
